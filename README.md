### Hearing Test Predictor

A simple Streamlit web app that predicts hearing issues based on age and physical test score using a logistic regression model.

---

### Live Demo

[Click here to try the app](https://vinayak251104-hearing-test-predictor.streamlit.app/)

### Run Locally

```bash
git clone https://github.com/vinayak251104/hearing_test_predictor.git
cd hearing_test_predictor
pip install -r requirements.txt
streamlit run Dashboard.py
